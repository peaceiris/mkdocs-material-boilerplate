# MathJax

- [PyMdown - Material for MkDocs](https://squidfunk.github.io/mkdocs-material/extensions/pymdown/#arithmatex-mathjax)



## Configure `mkdocs.yml`

```
markdown_extensions:
    - mdx_math:
        enable_dollar_delimiter: True
```



## Example code

```sh
$$
P\cdot Q = \|P\|\|Q\|\cos\alpha
$$
```



## Example rendering

$$
